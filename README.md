# План заняття.

1. Що таке css;

2. Підключення стилів:

- inline;
- embeded;
- external;

3. Селектори:

- універсальний селектор;
- селектор id;
- селектор елемента;
- селектор класу;
- селектор нащадка;
- селектор дитини;

4. Каскад стилів, специфічність, наслідування;

5. Псевдокласи стану (:hover, :focus, :active).
